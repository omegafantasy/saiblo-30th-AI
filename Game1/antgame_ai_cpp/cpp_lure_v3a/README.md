# cpp_lure_v3a

`cpp_lure_v3a` 是 `cpp_lure_v3` 的激进抗性测试变体，不是当前最优解。

行为差异只有一处：

- 己方当前金币 `>= 200`
- 己方基地生成蚂蚁等级 `< 2`
- 在 v3 原本操作之前，优先尝试追加 `UpgradeGeneratedAnt`

该操作仍经过 `PublicState::can_apply_operation()` 过滤，因此只有合法且金币足够时才会实际发送。其余决策全部复用当前 v3。

2026-04-29 的座位平衡 32 局 `v3-a vs v3` 结果为 v3-a `1-31`，总血量差 `-781`。这个变体主要用于观察当前 v3 对 25 血蚂蚁的抗性，不作为提交最优解。
