# Root C++ AI (ai_v1)

该目录是独立于 `Ant-Game` 仓库组织的 C++ AI 源码与构建入口。

## 编译

```bash
cd /www/ai_cpp/v1
make
```

生成可执行文件：`/www/ai_cpp/v1/ai_v1`

## 协议

- 输入：逐行文本
  - `"<seat> <seed>"`
  - 单行 JSON 状态
- 输出：`4 字节大端长度 + 文本命令串`（末行必须 `8`）

## 说明

- 该 AI 已迁移并复用 `past_AIs/Generals-AI` 的核心思路：
  - 敌方威胁图（threat map）估计。
  - 多步军队行动（按科技层级的 2/3/5 步预算）。
  - 主将安全阈值控制（避免过度抽空主将格）。
  - 经济行为（主将升级、科技升级、前线征召副将）。
  - 基于目标价值的攻占优先级（敌主将 > 敌副将 > 中立副将/农夫 > 普通格）。
- 仍是启发式策略，不包含深度博弈搜索。
- 根目录脚本 `eval_cpp_local.py` 与 `ai_cpp_policy.py` 默认使用本可执行文件。
